BiclusterMixin
[source]
#
Mixin class for all bicluster estimators in scikit-learn.
This mixin defines the following functionality:
biclusters_
property that returns the row and column indicators;
get_indices
method that returns the row and column indices of a bicluster;
get_shape
method that returns the shape of a bicluster;
get_submatrix
method that returns the submatrix corresponding to a bicluster.
